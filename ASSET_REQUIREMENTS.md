# Required Assets for Publishing

## 🖼️ Assets You Need to Upload

Upload these 4 files to the `assets/images/` folder:

### 1. App Icon (icon.png)
- **Size:** 1024 x 1024 pixels
- **Format:** PNG (no transparency)
- **Content:** Your app logo/icon design

### 2. Adaptive Icon (adaptive-icon.png)
- **Size:** 1024 x 1024 pixels
- **Format:** PNG
- **Content:** Same as app icon (Android version)

### 3. Splash Screen (splash.png)
- **Size:** 1284 x 2778 pixels (iPhone 14 Pro Max)
- **Format:** PNG
- **Content:** Your existing splash image
- **Source:** https://imgur.com/a/cultivate-splash-screen-pm3nvi8

### 4. Favicon (favicon.png)
- **Size:** 48 x 48 pixels
- **Format:** PNG
- **Content:** Small version of your app icon (for web)

## 🛠️ How to Create These

### Option 1: Use Your Splash Image
1. Download your splash image from the Imgur link
2. Save as `splash.png` (resize to 1284x2778 if needed)
3. Crop/simplify it to create the app icon (1024x1024)
4. Use the same design for adaptive-icon.png
5. Shrink the icon to 48x48 for favicon.png

### Option 2: Free Tools
- **Canva:** Easy templates and resizing
- **Figma:** Professional design tool (free)
- **Online:** Use any image resizer tool

## 📁 File Structure
```
assets/
└── images/
    ├── icon.png          (1024x1024)
    ├── adaptive-icon.png (1024x1024)
    ├── splash.png        (1284x2778)
    └── favicon.png       (48x48)
```
